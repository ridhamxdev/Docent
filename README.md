## Overview

**Docent** is a professional platform designed to bridge the gap between dental professionals, students, and patients. It serves as a unified ecosystem for networking, knowledge sharing, and professional growth within the dental community.

This project is being developed in collaboration with **Dr. Ashay Jain (BDA)**, ensuring that the platform meets the real-world needs of dental practitioners and students.

## Tech Stack

- **Frontend (Mobile)**: React Native, Expo, NativeWind (TailwindCSS)
- **Frontend (Web/Admin)**: Next.js, React
- **Backend**: Node.js, Express
- **Database**: Firebase Firestore, DynamoDB (for messaging)
- **Authentication**: Firebase Auth

## Getting Started

### Prerequisites
- Node.js (v18+)
- Expo Go app on your mobile device

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/ridhamxdev/Docent.git
    cd Docent
    ```

2.  **Install dependencies**:
    ```bash
    cd mobile
    npm install
    ```

3.  **Run the application**:
    ```bash
    npx expo start
    ```

4.  **Scan the QR code**: Use the Expo Go app on your phone to scan the QR code and launch the app.
